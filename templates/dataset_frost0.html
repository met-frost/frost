Dataset documentation for time series type frost0

The frost0 dataset has the following structure: 
{
    "tstype": "frost0",
    "tseries": [
        {
            "header": {
                "id": {
                    "source": "...",
                    "sensorLevel": "...",
                    "element": "..."
                },
                "extra": {
                    "organizations": ["...", "...", ...],
                    "pos": {
                        "lon": "...",
                        "lat": "..."
                    }
                }
            },
            "observations": [
               {
                   "time": "...",
                   "body": {
                       "pos": {
                           "lon": "...",
                           "lat": "..."
                        },
                        "value": "...",
                        "quality": "..."
                    }
                },
                ...
            ]
        },
        ...
    ]
}
  • All properties of the header/id object are required.
  • The header/extra object is optional.
  • The organizations array and pos object are both optional in any header/extra object.
  • The observation time must be of the form yyyy-mm-ddThh:mm:ssZ
  • Only value is required in the observation body object.
  • Both lon and lat must be present in any pos object in the observation body object or a header/extra object.
