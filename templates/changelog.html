Changelog

Date
2026-02-09

Beta closed

The release candidate has some breaking changes compared to the removed beta version. If you are still using the beta version, please switch to the release candidate.

2025-11-20

New database back-end

Users of frost-rc may notice some changes in what data is available (due to changes in ingestion and authentication), but we hope it is mostly a performance improvement. Compared to frost-rc we are missing:

  • Aggregations (max, min, avg...)
  • Calculations for some specific parameters
2023-10-23

Authentication turned on

Authentication is now turned on, uses same basic syntax (and the user id) as in v0 frost. However bearer authentication will work differently for accessing restricted data.
2022-11-01

Beta version released

Release of beta frost v1. There are still a few kinks to work out, so please contact us if you believe you have encountered a bug.
